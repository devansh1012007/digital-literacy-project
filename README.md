## Project Overview
This repository serves as my **Digital Literacy Portfolio**, created as part of my role as a **Student Digital Ambassador**. The goal of this project is to showcase foundational skills in professional digital presence, online safety, collaborative tools, and effective communication.

The portfolio is organized into five distinct modules, each representing a key area of digital literacy:
1. **Module 1:** Digital Literacy Awareness 
2. **Module 2:** Professional Digital Presence 
3. **Module 3:** Coding & Collaboration Platforms 
4. **Module 4:** Professional Communication & Etiquette 
5. **Module 5:** Cybercrime Awareness & Prevention 

---

## Module Summaries & Deliverables

### 🖥️ Task 1: Awareness Infographic
* **Description:** A one-page visual resource explaining digital literacy, safe practices, and email etiquette.
* **Tools Used:** Canva
* **Folder:** `task-1-presentation/`
